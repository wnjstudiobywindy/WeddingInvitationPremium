FAQ
Apakah undangan bisa dibuka di semua HP?
+
Ya! Undangan kami berbasis HTML yang bisa dibuka di browser HP manapun — iPhone, Android, semua bisa. Tidak perlu install aplikasi apapun.
Berapa lama proses pengerjaan?
+
Standar kami adalah 24 jam setelah data lengkap diterima. Untuk Paket VIP bisa lebih cepat dengan layanan priority.
Apakah perlu hosting berbayar?
+
Tidak harus! File HTML bisa dibagikan langsung via WA tanpa hosting. Atau bisa gratis hosting di Netlify. Kalau mau domain sendiri (nama.com), itu pilihan Paket VIP.
Bisa custom warna & tema sendiri?
+
Untuk Paket Basic & Premium, warna mengikuti tema yang dipilih. Untuk custom penuh — warna, font, layout — tersedia di Paket VIP Custom.
Bagaimana cara pembayaran?
+
Transfer via BCA, BNI, Mandiri, GoPay, OVO, atau DANA. Pembayaran DP 50% di awal, pelunasan setelah file siap dan disetujui.
Berapa maksimal revisi?
+
Basic: 1x revisi. Premium: 2x revisi. VIP Custom: unlimited revisi hingga sesuai keinginan.
